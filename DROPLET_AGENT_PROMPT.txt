TASK: Bootstrap Fish Speech S2 API Server on DigitalOcean H200 GPU Droplet

YOU ARE: Already SSH'd into droplet as root at 162.243.235.203

WHAT TO DO:

0. Clone voicebox-api repo:
   cd /opt
   rm -rf voicebox-api
   git clone https://github.com/pc-style/voicebox-api.git voicebox-api
   cd voicebox-api

1. Install base packages:
   apt-get update -y
   apt-get install -y git curl wget python3 python3-pip build-essential
   apt-get install -y docker.io
   systemctl start docker

2. Add NVIDIA Docker support:
   distribution=$(. /etc/os-release;echo $ID$VERSION_ID)
   curl -s -L https://nvidia.github.io/nvidia-docker/gpgkey | apt-key add -
   curl -s -L https://nvidia.github.io/nvidia-docker/$distribution/nvidia-docker.list | tee /etc/apt/sources.list.d/nvidia-docker.list
   apt-get update && apt-get install -y nvidia-container-toolkit
   systemctl restart docker

3. Clone Fish Speech repo:
   cd /opt
   rm -rf fish-speech
   git clone https://github.com/fishaudio/fish-speech.git fish-speech
   cd fish-speech

4. Create docker-compose override:
   cat > docker-compose.override.yml <<'EOF'
version: '3.8'
services:
  server:
    runtime: nvidia
    ports:
      - "0.0.0.0:8080:8080"
    environment:
      FISH_API_BIND: "0.0.0.0"
      FISH_API_PORT: "8080"
      CUDA_VISIBLE_DEVICES: "0"
EOF

5. Create directories:
   mkdir -p checkpoints/s2-pro references data

6. Copy reference audio and English texts from voicebox-api:
   cp /opt/voicebox-api/assets/reference.mp3 /opt/fish-speech/references/
   cp /opt/voicebox-api/assets/reference.txt /opt/fish-speech/references/
   unzip -q /opt/voicebox-api/assets/english-texts.zip -d /opt/fish-speech/references/

7. Download models (10-20 minutes):
   pip install -q huggingface_hub
   python3 << 'PYEOF'
from huggingface_hub import snapshot_download
snapshot_download(
  "fishaudio/s2-pro",
  repo_type="model",
  local_dir="/opt/fish-speech/checkpoints/s2-pro",
  allow_patterns=["*.pth", "*.json"],
  resume_download=True
)
print("Models downloaded!")
PYEOF

8. Start Docker container:
   cd /opt/fish-speech
   docker compose --profile server up -d --build

9. Wait 30 seconds, then verify API is running:
   curl http://127.0.0.1:8080/v1/health

10. Check GPU is detected:
   docker compose exec server nvidia-smi

11. Setup firewall:
    apt-get install -y ufw
    ufw --force enable
    ufw default deny incoming
    ufw default allow outgoing
    ufw allow 22/tcp
    ufw allow 8080/tcp

SUCCESS INDICATORS:
- curl http://127.0.0.1:8080/v1/health returns {"status": "ok"}
- docker compose ps shows 'server' container running
- nvidia-smi shows H200 GPU with 141GB VRAM
- UFW status shows rules enabled
